# 🎯 ビンゴメーカー (Bingo Maker)

カスタマイズ可能なビンゴカードを作成・共有できるWebアプリケーションです。DDRなどの音楽ゲームの目標管理に最適！

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Language](https://img.shields.io/badge/language-HTML%2FCSS%2FJS-orange.svg)

## ✨ 特徴

- **3つのサイズ対応**: 3×3、4×4、5×5から選択可能
- **完全カスタマイズ**: 各マスにタイトル、目標、難易度（★1〜5）を設定
- **3段階の状態管理**: 未着手 → 進行中 → 達成
- **ビンゴライン計算**: 縦・横・斜めの達成ラインを自動カウント
- **共有機能**: URLリンクで他のユーザーとビンゴを共有
- **画像保存**: ビンゴカードをJPG画像としてダウンロード
- **シャッフル機能**: マスをランダムに並び替え（元に戻すことも可能）
- **達成記録**: 達成履歴を自動・手動で記録
- **多言語対応**: 日本語・中文・English
- **オフライン対応**: ローカルストレージに保存、サーバー不要

## 🚀 デモ

GitHub Pagesでホスティング後、以下のURLでアクセスできます：

```
https://[your-username].github.io/[repo-name]/bingo-public.html
```

## 📖 使い方

### ビンゴの作成

1. サイズ（3×3、4×4、5×5）を選択
2. 6桁のコードが自動生成されます
3. 各マスをクリックして内容を編集

### マスの編集

- **左クリック**: 状態を切り替え（未着手 → 進行中 → 達成）
- **右クリック**: 編集画面を開く
  - タイトル（曲名など）
  - 目標（スコア、フルコンなど）
  - 難易度（★1〜5）

### 機能ボタン

| ボタン | 機能 |
|--------|------|
| 📤 共有 | 共有用URLをクリップボードにコピー |
| 📷 画像保存 | ビンゴカードをJPG画像として保存 |
| 🔀 シャッフル | マスをランダムに並び替え |
| ↩️ 元に戻す | シャッフル前の順序に戻す |
| 🗑️ 全リセット | 全マスの状態を未着手に戻す |

### ビンゴの読み込み

- 6桁のコードを入力して「読込」をクリック
- または共有URLを開く

## 🛠️ 技術仕様

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
- **データ保存**: localStorage（ブラウザ内保存）
- **画像生成**: html2canvas
- **サーバー**: 不要（静的ファイルのみ）

## 🌐 GitHub Pagesへのデプロイ

1. このリポジトリをフォークまたはクローン
2. GitHub リポジトリの Settings → Pages
3. Source を「main branch」に設定
4. 数分後にURLが発行されます

## ⚠️ 注意事項

- データはブラウザのlocalStorageに保存されます
- **7日間アクセスがないデータは自動削除**されます
- 異なるブラウザ・デバイス間でデータは共有されません
- 大切なビンゴは共有URLで保存することをおすすめします
